## DEB-package automation

`./cmd` - директория с исходным кодом `.go` и `.sh`.
 - `.go` - компилируем бинарники с именем папки. Складываем в `${TMP_DIR}` 
 - `.sh` - копируем в `${TMP_DIR}`


Сборка DEB-пакета осуществляется nsfp-docker.
